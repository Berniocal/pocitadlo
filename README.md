# Počítadlo – PWA

## Nastavení

1. Ve Firebase Console vytvořte projekt.
2. Přidejte **Web app**.
3. V Authentication → Sign-in method povolte **Google**.
4. V Realtime Database vytvořte databázi.
5. Konfiguraci webové aplikace vložte do `config.js`.
6. V `config.js` změňte oba povolené e-maily.
7. Stejné e-maily změňte v `database.rules.json` a pravidla vložte do Realtime Database → Rules.
8. Nahrajte celý obsah na HTTPS hosting (Firebase Hosting nebo GitHub Pages).

Pro lokální test aplikaci neotevírejte jako soubor. Spusťte ji přes lokální HTTP server.
